# quizApp

## quizApp은
- 자신에 대한 퀴즈를 만들고 친구에게 공유하여 직접 풀 수 있고, 랭킹 정보가 남으며 문제를 공유받은 사람도 자신만의 퀴즈를 만들 수 있는 웹앱


## 사용 기술 스택 및 라이브러리
- React
- Redux
- Kakao SDK
- Firebase Firestore
- react-router-dom
- styled-components
- redux-thunk
- firebase-tools
- react-bootstrap
- react-adsense
- react-copy-to-clipboard
- react-share
